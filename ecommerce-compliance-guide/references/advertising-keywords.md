# 广告法禁用词/极限词清单

## 绝对化用语（高风险）

### 国家级/最高级/最佳类
- 国家级、最高级、最佳、最优、最好、最大、最高、最低、最、第一、唯一、顶级、极品、终极、极致

### 排名/地位类
- 第一品牌、NO.1、TOP1、排名第一、遥遥领先、领袖品牌、领导品牌、王者、冠军

### 程度类
- 绝对、完美、100%、百分百、万能、永久、永远、无敌、万能药、特效

## 特殊商品禁用词

### 医疗/药品/医疗器械
- 疗效最佳、药到病除、根治、安全无副作用、100%有效、治愈、痊愈、祖传秘方

### 保健食品
- 不得涉及疾病预防、治疗功能
- 不得声称具有保健功能以外的其他功能

### 化妆品
- 速效、速效美白、速效祛斑、强效、特效、换肤、去除皱纹、绝对安全

## 诱导性用语
- 点击领奖、恭喜获奖、全民免单、点击有惊喜、点击获取、点击转身
- 秒杀、抢爆、再不抢就没了、错过就没机会了、万人疯抢

## 虚假宣传相关
- 销量第一、全网第一、全网首发、全网独家
- 100%正品、假一赔万、假一罚十（如无实际赔付能力证明）

## 可使用替代词

| 禁用词 | 替代建议 |
|---|---|
| 最好 | 优质、优良、出色 |
| 最便宜 | 实惠、性价比高、优惠 |
| 第一 | 行业领先、深受欢迎、备受青睐 |
| 绝对 | 相当、非常、确实 |
| 永久 | 持久、长效、稳定 |
| 万能 | 多功能、适用范围广 |

## 注意事项

1. 以上清单并非穷尽，需结合具体语境判断
2. 即使不使用上述词汇，如整体表述构成虚假宣传仍属违法
3. 数据宣称（销量第一、市场占有率等）需有权威数据来源支撑
4. 定期关注市场监管总局发布的典型案例和执法指南
