# 直播营销合规话术指引

## 基本原则

1. **真实准确：** 直播中对商品或服务的介绍应当真实、准确，不得虚假宣传
2. **可识别性：** 直播营销构成商业广告的，应当显著标明"广告"
3. **未成年人保护：** 不得利用未成年人进行直播营销，不得向未成年人推销不适宜商品
4. **数据真实：** 不得虚构交易、成交量、用户评价等数据

## 主播义务

### 身份与资质
- 直播营销人员应当年满16周岁
- 从事医疗、药品、医疗器械、保健食品直播营销的，应当由具备相应资质的人员进行
- 主播为商家员工的，应当表明身份；为代言人的，应当符合广告代言人要求

### 内容合规
- 不得对商品进行虚假宣传，欺骗、误导消费者
- 不得发布虚假或引人误解的信息
- 不得贬低其他生产经营者的商品或服务
- 不得侵犯他人知识产权

## 禁用话术

### 绝对禁止
- "全网最低价"（无法证实）
- "假一赔万"（超出实际赔付能力）
- "医院同款"（医疗商品需资质）
- "医生推荐"（医疗相关需资质）
- "吃了就能瘦/ curing 百病"（功效宣称需依据）

### 高风险话术
- "限量XX件，抢完即止"（实际库存充足）
- "原价XXX，今天只要XX"（原价虚假）
- "全网销量第一"（无权威数据来源）
- "明星同款"（未经授权）

### 需谨慎使用
- "推荐"、"好物分享"（可能构成广告，需标明）
- "亲测有效"（个人体验不能替代产品功效证明）
- "粉丝福利"（如实际为商业推广，需明示）

## 特殊商品直播要求

### 食品
- 不得声称具有保健功能以外的其他功能
- 不得对普通食品声称具有疾病预防、治疗功能
- 需展示食品标签信息

### 化妆品
- 不得宣称医疗作用
- 不得使用"速效"、"换肤"等夸大用语
- 特殊化妆品需展示批准文号

### 保健食品
- 必须声明"保健食品不是药物，不能代替药物治疗疾病"
- 不得涉及疾病预防、治疗功能

## 促销活动合规

### 限时促销
- 明确标注促销起止时间
- 不得虚构"倒计时"制造紧迫感
- 促销结束后恢复原价应当真实

### 抽奖/赠品
- 抽奖式有奖销售最高奖金额不得超过5万元
- 赠品信息应当真实，不得虚构
- 明确赠品获取条件和方式

## 数据合规

### 禁止行为
- 刷单炒信（虚构交易、成交量）
- 删除/屏蔽消费者评价
- 虚构用户评价
- 篡改用户评价

### 合规做法
- 真实展示商品销量和评价
- 建立评价审核机制（仅删除违法违规内容）
- 保存直播视频不少于三年

## 应急处理

### 直播中出现合规问题
1. **立即纠正：** 发现话术违规，立即更正并澄清
2. **下架商品：** 如商品本身存在问题，立即下架
3. **记录存档：** 保存相关直播录像和记录
4. **报告处理：** 如涉及严重违规，报告法务/合规部门

### 消费者投诉处理
1. 建立直播投诉快速响应机制
2. 明确投诉处理时限（建议24小时内响应）
3. 重大投诉升级至法务/合规部门
