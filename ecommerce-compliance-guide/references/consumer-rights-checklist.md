# 消费者权益保护检查清单

## 一、商品信息展示

### 基本信息
- [ ] 商品名称、规格、产地、生产日期、保质期/安全使用期
- [ ] 价格（含运费、税费说明）
- [ ] 生产者/销售者名称、地址、联系方式
- [ ] 售后服务政策

### 特殊标注
- [ ] 进口商品中文标识
- [ ] 限期使用商品生产日期和保质期
- [ ] 3C认证产品认证标志和编号
- [ ] 不适用七天无理由退货的商品明确标注并提示确认

## 二、七天无理由退货

### 退货条件
- [ ] 消费者签收商品之日起七日内
- [ ] 商品完好（不影响二次销售）
- [ ] 消费者承担退货运费（另有约定除外）

### 不适用退货的商品（需消费者确认）
- [ ] 消费者定作的商品
- [ ] 鲜活易腐商品
- [ ] 在线下载或拆封的数字化商品
- [ ] 交付的报纸、期刊
- [ ] 拆封后影响人身安全或健康的商品
- [ ] 拆封后易导致品质改变的商品
- [ ] 激活/试用后价值贬损较大的商品

### 退货流程
- [ ] 提供便捷的退货申请入口
- [ ] 明确退货地址和联系方式
- [ ] 收到退货后及时验收
- [ ] 符合退货条件的及时退款

## 三、个人信息保护

### 收集环节
- [ ] 公开隐私政策，明示处理规则
- [ ] 收集前取得消费者同意
- [ ] 不超出必要范围收集信息
- [ ] 敏感信息取得单独同意

### 使用环节
- [ ] 按约定目的使用信息
- [ ] 不非法出售或提供给他人
- [ ] 采取必要安全措施

### 权利保障
- [ ] 提供查询、更正、删除渠道
- [ ] 响应消费者行使权利的请求
- [ ] 建立信息泄露应急响应机制

## 四、售后服务

### 投诉渠道
- [ ] 提供在线投诉入口
- [ ] 提供投诉电话
- [ ] 提供投诉邮箱
- [ ] 公示投诉处理时限

### 处理要求
- [ ] 及时受理投诉
- [ ] 调查核实情况
- [ ] 反馈处理结果
- [ ] 保存处理记录

## 五、争议解决

### 协商和解
- [ ] 积极与消费者协商
- [ ] 提出合理解决方案

### 调解
- [ ] 配合消费者协会调解
- [ ] 配合市场监管部门调解

### 仲裁/诉讼
- [ ] 尊重消费者选择权
- [ ] 配合仲裁或诉讼程序

## 六、禁止行为

- [ ] 不得发布虚假或引人误解的商品信息
- [ ] 不得拒绝或拖延履行七天无理由退货义务
- [ ] 不得删除或屏蔽消费者评价
- [ ] 不得未经同意收集、使用消费者个人信息
- [ ] 不得向未成年人销售不适宜商品
- [ ] 不得利用格式条款排除或限制消费者权利

## 七、记录保存

- [ ] 商品和服务信息保存不少于三年
- [ ] 交易信息保存不少于三年
- [ ] 消费者投诉处理记录保存不少于三年
- [ ] 直播视频保存不少于三年
