# 中国电子商务合规指南

## 适用对象

- 综合电商平台运营方（淘宝、京东、拼多多等）
- 社交电商/直播电商平台（抖音、快手、小红书等）
- 平台内经营者（商家）
- 电商企业法务/合规团队

## 法规框架（A层 - 硬规则）

| 法律/法规 | 发布机关 | 核心适用范围 |
|---|---|---|
| 《中华人民共和国电子商务法》 | 全国人大常委会 | 电商经营者定义、平台责任、数据报送 |
| 《中华人民共和国广告法》 | 全国人大常委会 | 广告内容准则、代言规范、虚假广告 |
| 《中华人民共和国消费者权益保护法》 | 全国人大常委会 | 七天无理由退货、个人信息保护、欺诈赔偿 |
| 《中华人民共和国产品质量法》 | 全国人大常委会 | 产品质量责任、缺陷产品召回 |
| 《中华人民共和国食品安全法》 | 全国人大常委会 | 食品经营许可、特殊食品监管 |
| 《中华人民共和国个人信息保护法》 | 全国人大常委会 | 个人信息处理规则、跨境传输 |
| 《网络交易监督管理办法》 | 国家市场监督管理总局 | 网络交易行为规范、平台义务 |
| 《规范促销行为暂行规定》 | 国家市场监督管理总局 | 促销活动规则、价格欺诈 |
| 《网络直播营销管理办法（试行）》 | 国家网信办等七部门 | 直播营销机构、主播、平台责任 |
| 《互联网广告管理办法》 | 国家市场监督管理总局 | 互联网广告识别、弹窗规则 |

## 模块一：商品与服务合规

### 1.1 商品信息合规

**法律依据：** 《电子商务法》第17条、《消费者权益保护法》第20条、《产品质量法》第27条

**合规要点：**
- 商品信息应当真实、准确、完整
- 不得作虚假或引人误解的宣传
- 应当明示商品的主要成分、产地、生产日期、有效期限等信息
- 强制性认证产品（CCC）必须标注认证标志

**风险评级：**
- 🔴 明确违法：销售假冒伪劣商品、伪造产地
- 🟡 高风险：商品信息与实际严重不符、隐瞒重要信息
- 🟠 需证据支持：商品功效宣称（需提供检测报告或备案凭证）
- 🔵 表述不清：商品描述存在歧义

**操作建议：**
1. 建立商品信息审核机制，上架前审核
2. 对商家提交的商品信息进行抽查
3. 建立消费者投诉快速处理通道
4. 对高风险类目（食品、化妆品、医疗器械）实行更严格审核

**检查清单：**
- [ ] 商品页面是否包含完整的商品信息
- [ ] 是否明示价格、运费、售后服务
- [ ] 是否标注生产者名称、地址
- [ ] 限期使用商品是否标注生产日期和安全使用期
- [ ] 进口商品是否标注中文标识

### 1.2 产品质量责任

**法律依据：** 《产品质量法》第40-43条、《电子商务法》第38条

**合规要点：**
- 平台对关系消费者生命健康的商品/服务负有资质审核义务
- 明知或应知商家销售不合格商品未采取必要措施的，承担连带责任
- 建立缺陷产品召回配合机制

**风险评级：**
- 🔴 明确违法：销售明知不合格的商品、拒绝配合召回
- 🟡 高风险：未履行资质审核义务导致不合格商品流入
- 🟠 需证据支持：商家声称的质检报告真实性

**操作建议：**
1. 对食品、药品、医疗器械等特殊商品实行准入资质审查
2. 建立商品抽检机制
3. 对消费者集中投诉的商品启动调查
4. 配合市场监管部门的缺陷产品调查

### 1.3 价格合规

**法律依据：** 《价格法》第14条、《规范促销行为暂行规定》

**合规要点：**
- 明码标价，不得利用虚假或使人误解的价格手段诱骗交易
- 促销时标注的原价应当真实有据
- 不得虚构原价、虚假优惠折价

**风险评级：**
- 🔴 明确违法：虚构原价、先涨价后打折
- 🟡 高风险：促销规则不透明、附加不合理条件
- 🔵 表述不清：价格标注方式容易引起误解

**操作建议：**
1. 建立促销价格审核机制
2. 保留价格变动记录备查
3. 促销规则应当清晰明示，不得设置不合理门槛

## 模块二：广告与营销合规

### 2.1 广告内容准则

**法律依据：** 《广告法》第4-12条、《互联网广告管理办法》

**合规要点：**
- 广告不得含有虚假或引人误解的内容
- 不得使用"国家级"、"最高级"、"最佳"等绝对化用语
- 医疗、药品、医疗器械、保健食品广告不得含有表示功效、安全性的断言或保证
- 互联网广告应当具有可识别性，显著标明"广告"

**风险评级：**
- 🔴 明确违法：虚假广告、使用绝对化用语、未经审查发布特殊商品广告
- 🟡 高风险：夸大商品功效、使用极限词
- 🟠 需证据支持：数据引用（需提供数据来源）
- 🔵 表述不清：广告与信息的边界模糊

**操作建议：**
1. 建立广告发布前审核机制
2. 对商家自行发布的商品描述进行关键词监控
3. 对直播带货话术建立合规指引
4. 建立广告投诉快速处理机制

**检查清单：**
- [ ] 是否显著标明"广告"（如属于广告）
- [ ] 是否含有绝对化用语
- [ ] 是否含有虚假或夸大内容
- [ ] 特殊商品广告是否已取得审查批准
- [ ] 是否侵犯他人知识产权

### 2.2 直播营销合规

**法律依据：** 《网络直播营销管理办法（试行）》

**合规要点：**
- 直播间运营者、直播营销人员应当年满16周岁
- 直播营销人员发布的商品或服务内容构成商业广告的，应当履行广告发布者/经营者/代言人的责任和义务
- 不得对商品进行虚假宣传，欺骗、误导消费者
- 应当建立健全未成年人保护机制

**风险评级：**
- 🔴 明确违法：虚假宣传、销售假冒伪劣商品、数据造假（刷单炒信）
- 🟡 高风险：主播未履行审核义务、直播间信息不真实
- 🟠 需证据支持：直播中的数据宣称（销量、用户评价等）

**操作建议：**
1. 建立主播/直播间准入审核机制
2. 对直播内容进行实时监控或回放抽查
3. 建立直播商品信息审核机制
4. 保存直播视频不少于三年

### 2.3 促销活动合规

**法律依据：** 《规范促销行为暂行规定》

**合规要点：**
- 促销规则应当清晰明示，不得设置不合理条件
- 抽奖式有奖销售最高奖金额不得超过5万元
- 不得虚构奖品或故意让内定人员中奖
- 促销期间不得先涨价再降价

**风险评级：**
- 🔴 明确违法：抽奖最高奖超过5万元、虚假有奖销售
- 🟡 高风险：促销规则不透明、附加不合理条件
- 🔵 表述不清：促销规则存在歧义

## 模块三：消费者权益保护

### 3.1 七天无理由退货

**法律依据：** 《消费者权益保护法》第25条、《网络购买商品七日无理由退货暂行办法》

**合规要点：**
- 消费者有权自收到商品之日起七日内退货（部分商品除外）
- 不适用七天无理由退货的商品应当经消费者确认
- 退货商品应当完好，消费者承担退货运费（另有约定除外）

**不适用七天无理由退货的商品：**
- 消费者定作的商品
- 鲜活易腐的商品
- 在线下载或消费者拆封的音像制品、计算机软件等数字化商品
- 交付的报纸、期刊
- 拆封后易影响人身安全或生命健康的商品（如食品、药品、化妆品）
- 拆封后易导致商品品质发生改变的商品
- 一经激活或试用后价值贬损较大的商品

**风险评级：**
- 🔴 明确违法：拒绝履行七天无理由退货义务、擅自扩大不适用范围
- 🟡 高风险：退货流程设置不合理障碍
- 🔵 表述不清：退货规则表述不清导致争议

**操作建议：**
1. 在商品页面显著位置标注是否支持七天无理由退货
2. 建立便捷的退货申请和处理流程
3. 对不适用退货的商品设置确认环节
4. 不得无故拒绝或拖延退货

### 3.2 个人信息保护

**法律依据：** 《个人信息保护法》、《电子商务法》第23-25条

**合规要点：**
- 处理个人信息应当具有明确、合理的目的
- 应当公开个人信息处理规则，明示处理目的、方式和范围
- 取得个人同意（敏感个人信息需单独同意）
- 不得过度收集个人信息
- 采取必要措施保障个人信息安全

**风险评级：**
- 🔴 明确违法：未经同意收集/使用个人信息、非法出售个人信息
- 🟡 高风险：过度收集、未明示处理规则
- 🟠 需证据支持：取得同意的记录保存

**操作建议：**
1. 制定并公开隐私政策
2. 建立个人信息分类分级管理制度
3. 对商家收集消费者信息的行为进行规范
4. 建立个人信息安全事件应急响应机制

**检查清单：**
- [ ] 是否制定并公开隐私政策
- [ ] 收集个人信息是否取得同意
- [ ] 是否超出必要范围收集信息
- [ ] 是否建立信息安全保护措施
- [ ] 是否建立信息泄露应急响应机制

### 3.3 售后服务与争议解决

**法律依据：** 《消费者权益保护法》第39条、《电子商务法》第58-61条

**合规要点：**
- 建立便捷、有效的投诉、举报机制
- 公示投诉处理方式，及时处理并反馈
- 鼓励建立在线争议解决机制
- 保存商品和服务信息、交易信息不少于三年

**风险评级：**
- 🔴 明确违法：拒绝处理消费者投诉、删除消费者评价
- 🟡 高风险：拖延处理投诉、未建立有效投诉渠道

**操作建议：**
1. 建立多渠道投诉受理机制（在线、电话、邮件）
2. 明确投诉处理时限并公示
3. 建立消费纠纷先行赔付机制（鼓励）
4. 配合消费者协会和监管部门调解

## 模块四：平台责任与义务

### 4.1 平台经营者义务

**法律依据：** 《电子商务法》第27-46条

**核心义务：**
1. **身份核验：** 对申请进入平台销售商品或提供服务的经营者进行核验、登记
2. **信息报送：** 向市场监管部门报送平台内经营者身份信息
3. **规则公示：** 制定平台服务协议和交易规则，显著位置公示
4. **信用评价：** 建立健全信用评价制度，公示信用评价规则
5. **知识产权保护：** 建立知识产权保护规则，采取必要措施
6. **安全保障：** 采取技术措施和其他必要措施保证网络安全、稳定运行

**风险评级：**
- 🔴 明确违法：未履行身份核验义务、未报送信息、对违法经营行为未采取必要措施
- 🟡 高风险：平台规则不公平不合理、未显著公示规则变更

### 4.2 数据报送义务

**法律依据：** 《电子商务法》第25条、28条

**合规要点：**
- 向有关部门报送平台内经营者身份信息
- 提示未办理市场主体登记的经营者依法办理登记
- 配合市场监管部门查处违法经营行为，提供必要数据

## 使用说明

### 新建指南 (`--build`)
运行本技能，根据提示选择需要包含的模块，生成完整合规指南。

### 审查现有规则 (`--review`)
上传现有平台规则/商家协议，本技能将对照现行法律法规进行合规性审查，标注风险点。

### 更新指南 (`--update`)
当法律法规发生变化时，运行本技能更新对应模块，保留原有框架。

## 重要声明

本指南基于截至 [日期] 的现行法律法规编制。法律法规可能发生变化，建议：
1. 定期（建议每季度）复核指南内容
2. 关注市场监管总局、网信办等部门最新政策
3. 重大合规决策前咨询专业法律顾问
4. 本指南不构成法律意见，具体案件应寻求专业法律服务
