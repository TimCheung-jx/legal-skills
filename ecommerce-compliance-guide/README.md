# 电商合规指南技能 (Ecommerce Compliance Guide)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![OpenClaw](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)

> 中国电子商务合规指南构建器。基于《电子商务法》《广告法》《消费者权益保护法》等法律法规，为电商企业法务/合规团队生成结构化合规指南。

## 功能特性

- **法规框架全覆盖**：涵盖电商经营核心法律法规
- **模块化合规指南**：商品与服务、广告营销、消费者权益三大模块
- **风险评级体系**：红/黄/橙/蓝四级风险标识
- **实用检查清单**：可直接使用的合规检查表
- **多场景支持**：综合电商、社交电商、直播电商

## 适用场景

- 构建/更新电商合规手册
- 审查平台规则合规性
- 培训合规团队
- 日常合规自查

## 法规框架

| 法律/法规 | 核心适用范围 |
|:---|:---|
| 《电子商务法》 | 电商经营者定义、平台责任、数据报送 |
| 《广告法》 | 广告内容准则、代言规范、虚假广告 |
| 《消费者权益保护法》 | 七天无理由退货、个人信息保护、欺诈赔偿 |
| 《产品质量法》 | 产品质量责任、缺陷产品召回 |
| 《食品安全法》 | 食品经营许可、特殊食品监管 |
| 《个人信息保护法》 | 个人信息处理规则、跨境传输 |
| 《网络交易监督管理办法》 | 网络交易行为规范、平台义务 |
| 《网络直播营销管理办法》 | 直播营销机构、主播、平台责任 |
| 《互联网广告管理办法》 | 互联网广告识别、弹窗规则 |

## 使用方法

安装后，使用以下命令触发：

```bash
# 新建合规指南
/ecommerce-compliance-guide --build

# 审查现有规则
/ecommerce-compliance-guide --review

# 更新合规指南
/ecommerce-compliance-guide --update
```

## 合规模块

### 模块一：商品与服务合规

- **商品信息合规**：信息真实性、完整性、认证标志
- **产品质量责任**：资质审核、缺陷产品召回
- **价格合规**：明码标价、促销规则、价格欺诈防范

### 模块二：广告与营销合规

- **广告内容准则**：真实性、合法性、导向要求
- **代言与推荐**：代言人责任、直播营销规范
- **促销活动**：限时折扣、满减规则、赠品规范

### 模块三：消费者权益保护

- **七天无理由退货**：适用范围、除外商品、运费承担
- **个人信息保护**：收集规则、使用限制、安全保障
- **售后服务**：三包责任、投诉处理、争议解决

## 风险评级说明

| 等级 | 定义 | 处置建议 |
|:---|:---|:---|
| 🔴 **红色** | 明确违法：违反法律强制性规定 | 立即停止，整改后方可继续 |
| 🟡 **黄色** | 高风险：存在重大合规隐患 | 限期整改，加强监控 |
| 🟠 **橙色** | 需证据支持：需要检测报告或备案 | 补充材料，完善证据链 |
| 🔵 **蓝色** | 表述不清：存在歧义或不够明确 | 优化表述，提高清晰度 |

## 输出格式

每个合规模块包含：
- **法律依据**：具体法律条款
- **合规要点**：核心要求总结
- **风险评级**：四级风险标识
- **操作建议**：具体实施步骤
- **检查清单**：可直接使用的检查表

## 示例

**输入**：
```
/ecommerce-compliance-guide --build
```

**输出**：
```
# 中国电子商务合规指南

## 模块一：商品与服务合规

### 1.1 商品信息合规

**法律依据：** 《电子商务法》第17条、《消费者权益保护法》第20条

**合规要点：**
- 商品信息应当真实、准确、完整
- 不得作虚假或引人误解的宣传
- 应当明示商品的主要成分、产地、生产日期等信息

**风险评级：**
- 🔴 明确违法：销售假冒伪劣商品、伪造产地
- 🟡 高风险：商品信息与实际严重不符
...

**检查清单：**
- [ ] 商品页面是否包含完整的商品信息
- [ ] 是否明示价格、运费、售后服务
- [ ] 限期使用商品是否标注生产日期
...
```

## 适用平台类型

- **综合电商平台**：淘宝、京东、拼多多等
- **社交电商**：小红书、微信小商店等
- **直播电商**：抖音、快手等
- **垂直电商**：生鲜、母婴、美妆等

## 注意事项

- 本指南基于现行有效法律法规编制
- 法规更新时请及时调整合规要求
- 特殊行业（食品、药品、医疗器械）需遵守特别规定
- 建议定期组织合规培训和自查
